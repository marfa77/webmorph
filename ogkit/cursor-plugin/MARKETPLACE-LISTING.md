## Short description

Connect Cursor to OGKit’s hosted MCP server. Agents can list templates, build 1200×630 `og:image` URLs, emit Next.js `generateMetadata()` snippets, and validate live page OG tags — no local renderer.

## Long description

OGKit is a hosted Open Graph image API. This plugin bundles:

- **Remote MCP** at `https://www.webmorp.art/api/mcp` (Streamable HTTP, no API key for discovery tools)
- **Agent skill** for OG metadata workflows
- **Cursor rules** for `layout.tsx` / `page.tsx` metadata patterns

**Tools:** `og_list_templates`, `og_build_url`, `og_preview`, `og_nextjs_snippet`, `og_validate_page`, `ogkit_get_started`

**Free tier:** watermarked `demo=1` previews without signup. Sign in at webmorp.art for an auto-provisioned API key.
